# Images Directory

This directory is for storing image assets like:
- Hero images
- Portfolio project images  
- Client logos
- Team photos
- Icons

## Recommended Image Formats

- **WebP** - Best for modern browsers (smaller file size)
- **PNG** - For logos and images with transparency
- **JPG** - For photos

## Optimization Tips

- Use responsive images with `srcset`
- Add `loading="lazy"` attribute
- Compress images before uploading
- Recommended max width: 1920px for hero images
- Recommended max width: 800px for portfolio items

## Naming Convention

Use descriptive, lowercase names with hyphens:
- `hero-background.webp`
- `portfolio-project-1.webp`
- `client-logo-company-name.png`
